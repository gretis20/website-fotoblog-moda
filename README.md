# Fotoblog de Moda

## Descripción
Fotoblog de moda desarrollado como trabajo práctico, enfocado en el diseño editorial digital y la comunicación visual a través de imágenes y texto.
El sitio presenta contenidos inspirados en pasarelas internacionales, street style y tendencias de color, priorizando la estética, la composición visual y la experiencia del usuario.

## Contenido del sitio
- Artículos sobre tendencias de moda
- Referencias a pasarelas y street style
- Uso de imágenes editoriales
- Análisis de paletas de color (Pantone, Fashion Weeks)
- Diseño visual limpio y estructurado

## Estructura
El sitio cuenta con:
- Página de inicio con artículo principal
- Sección “Acerca de”
- Sección “Street Style”
- Sección de referencias
- Outfit Planner

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- Diseño responsive
- Navegación con menú hamburguesa

## Cómo visualizar el proyecto
1. Descargar o clonar el repositorio
2. Abrir el archivo `index.html` en un navegador web
3. Navegar por las distintas secciones desde el menú principal

## Objetivo del proyecto
Explorar la relación entre moda, color y diseño web, aplicando conceptos de diseño editorial y estética visual en un entorno digital.
